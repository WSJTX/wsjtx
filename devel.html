Operating Systems


WSJT-X, MAP65, and QMAP can be compiled for Windows, macOS, and Linux. Download links on the WSJT page will retrieve pre-built installation packages for each operating system. Source code is also available, along with instructions to help knowledgeable users build their own executables. Our code is copyrighted and licensed under terms of the GNU General Public License, GPLv3. Use of our software outside bounds of the GPLv3 license is prohibited, unless licensed by us under an alternative arrangement.

The graphical user interfaces (GUIs) for WSJT-X, MAP65, and QMAP are written in C++ and use the Qt framework. Most of the signal processing, encoding, and decoding is done in FORTRAN, and some functions are implemented in C.

Anyone interested in ham radio digital communication is invited to participate in further development of the programs. Source code is maintained in a public open-source repository at GitHub.
