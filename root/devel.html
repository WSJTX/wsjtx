Operating Systems


WSJT-X can be compiled for Windows, Linux, FreeBSD, macOS, and most other UNIX-like operating systems. The download links on pages for each program retrieve the available pre-compiled installation packages for Microsoft Windows, Linux, and macOS.  Users of other operating systems or distributions must compile the programs from source code. 

Source Code and License


WSJT-X, and MAP65 are open source programs.  Their code is copyrighted and licensed under terms of the GNU General Public License, GPLv3.

The graphical user interfaces (GUIs) for WSJT-X and MAP65 are written in C++ and use the Qt framework.  Most of the number-crunching in these programs is done in Fortran, and some functions are implemented in C.

Anyone interested in the communication problems addressed by these programs is invited to study, adapt, modify, or enhance the code and contribute to further development of the programs, while adhering to requirements of the GPLv3 license.  Source code is maintained in a public open-source repository at SourceForge. For updates and other information on the continuing development of these programs you are invited to subscribe to the WSJT developers email list.
